# Ray Tracing

Following the [Ray Tracing](https://github.com/RayTracing/raytracing.github.io) Book Series

## How to use

We support two configs, Debug and Release, defaulting to Debug if unspecified.

```bash
$ cmake -B build/<config> -DCMAKE_BUILD_TYPE=<config>
$ cmake --build build/<config>
$ ./bin/<config>/RayTracerInOneWeekend
```
